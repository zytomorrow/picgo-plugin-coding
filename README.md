# picgo-plugin-Coding

PicGo Uploader For Coding.net

### Install

```bash
npm i picgo-plugin-coding
```

### Usage

- 团队名称: coding.net团队名称
- 项目名称: 需要保存图片的库名
- token: coding.net's api token
- 分支: 需要存放图片的分支。默认master
- 存储文件夹: 需要存放照片的指定文件夹
